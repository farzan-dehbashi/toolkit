### Create database:
```
CREATE DATABASE <name>;
```
### Show databases:
```
SHOW DATABASES;
```
### Drop database:
```
DROP DATABASE <name>
```
### Switch between databases:
```
USE <name>
```
### pwd the database:
```
SELECT database();
```
